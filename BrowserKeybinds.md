# Chrome keybinds

- Toggle device toolbar - `M`
- Next panel - `Alt + —>`
- Previous Panel - `Alt + <—`
- Run Command - `Ctrl +  P`
- Show Computed Styles - `E`
- Show styles- `Q`
- Hide element - `H`
- Edit as HTML - `F2`

## Preferences

- Enable Ctrl + 1-9 shortcut to switch panels

## Elements

- Show rulers on hover
